Methods and Plan

To address the question specified, using logistic regression would be the best fit. Logistic regression is better for predicting binary values (0, 1 or true, false) which is needed to answer in the question specified. The project's goal is to guess the likelihood of someone subscribing to a newsletter which is either yes or no which suggests us to use logistic regression.

Assumptions

  • Each player's decision of subscribing to a newslettter should be independent and should not be affected by any other variables
  • The explanatory variables are not influenced by each other
  • Enough data to make predictions are provided

Potential limitations / weaknesses

  • The dataset may not be large enough for making highly accurate predictions
  • Outliers in the numerical values may affect performance

Comparing and Selecting the Model

  • Model performance will be evaluated using 3 factors. Accuracy, precision and recall.
  • Producing a confusion matrix to compare it with actual outcomes can help

Data Processing and Validation

  • Data will be split into 2 parts. 70% for training the data and 30% for testing the model
  • Then the model will be fitted in the training set
  • Lastly we will evaluate model's predictive capabilities using the remaining testing dataset
